# Barraponto Blog

Hosted at https://barraponto.blog.br
Deployed on Gitlab Pages using Jekyll.

## How to write

```
bundle exec draft "name of new draft"
bundle exec publish path/to/draft
```

## How to install

```
gem install bundler
bundle install
```

## How to update

```
bundle update github-pages
```

## How to deploy

```
git push gitlab master
```
